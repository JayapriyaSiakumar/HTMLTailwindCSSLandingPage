# Simple HTML Page with Tailwind CSS

This is a simple HTML page that uses Tailwind CSS for styling.

## Features

- Responsive design
- Simple and clean layout
- Easy to customize with Tailwind CSS

## Installation

1. Clone the repository
2. open the index.html file in a web browser

## Customization

To customize the page, you can edit the styles in the `tailwind.config.js` file.
